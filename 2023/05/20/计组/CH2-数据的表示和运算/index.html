      从最低位开始、按位相加、并往跟高位进位

      被减数不变、减数全部按位取反、末位 + 1, 减法变加法

# 标志位生成

ZF 对有无符号数都有意义
ZF = 1 (运算结果 nbit 全 0)
ZF 表示运算结果是否为 0

# 对有符号数

  1. OF (看溢出)
    OF = 1 时，说明发生溢出
    OF = 最高位进位⊕次高位进位 = Cout ⊕ Cn-1

  2. SF
    运算结果的正负性，0+1-
    SF = 最高本位和

# 对无符号数

  1. CF (看溢出)
    进位 / 借位标志，表无符号数的加减法是否发生了进位 or 借位。
    CF = 1, 说明发生了进位 / 借位，即发生了溢出
    CF = 最高位进位⊕Sub = Cout ⊕ Cin

    (sub = 1, 减法、sub = 0, 加法)

# 移位运算

# 算术移位

  1. 对原码
    符号位不变，
    对数值位、
    右移：高位补 0, 低位舍弃。舍弃位 = 0 -> ÷2; 舍弃位≠0 -> 丢失精度 2-n
    左移：低位补 0, 高位舍弃。舍弃位 = 0 -> ×2; 舍弃位≠0 -> 严重误差

  2. 对反码

      1. 正数：同原码
      1. 负数：
        左移右移：空位补 1, 多余舍弃

  3. 对补码

      1. 正数：同原码
      1. 负数：
        右移：高位补 1
        左移：低位补 0

# 逻辑移位

看作 "无符号数"
左移右移：空位补 0, 多余舍弃

# 循环移位

  1. 不带进位位：用移除的位补上空缺
  2. 带进位位：移出的位放到进位位上，原进位位补上空缺

# 浮点数的表示

浮点数真值：N = rE × M

  1. 阶码：
    E
    反映数值大小、浮点数的表示范围、小数点的实际位置；
    常用补码 or 移码表述的定点整数

  2. 尾数：
    M
    反映精度_数值部分的 n
    常用原码 or 补码表示的定点小数

=> 尾数给出了一个小数， 阶码指明了小数点向前 / 后移动了几位。

# 尾数的规格化

规定尾数的最高数值位必须是一个有效值。
1/R <= |M| < 1

# 左规

尾数出现 0.001
尾数 0.0xxxxxxxxx -----> 0.1xxxxxxxxxx...
左规：尾数算术左移、阶码减 1、直到尾数最高位是有效值。

# 右规

01.xxxxx
10.xxxxx
尾数出现溢出 (双符号位为 01、10)
右规：尾数算术右移、阶码加 1

# 特点

  1. 原码尾数规格化
    规格化的原码尾数，最高数值位一定是 1。

      1. 正数:
        0.1xxxx_______0.10000... ~ 0.1111111...
        尾数范围 M: 1/2 ~ ( 1-2-n)
      1. 负数:
        1.1xxxx_______1.1111111.. ~ 1.10000...
        尾数范围 M: -( 1-2-n)~ -1/2

  2. 补码尾数规格化
    规格化的补码尾数，符号位与最高数值位一定相反

      1. 正数:
        0.1xxxx________
        同原码正数范围
      1. 负数:
        1.0xxxx________1.00000... ~ 1.0111111...
        M: -1 ~ -( 1/2 + 2-n)
